# Check ARIMA viability for WARM components

Check ARIMA viability for WARM components

## Usage

``` r
.warm_arima_viable(x, min_n = 8L, sd_eps = 1e-08, min_unique = 3L)
```

## Arguments

- x:

  Numeric vector.

- min_n:

  Integer. Minimum length.

- sd_eps:

  Numeric. Minimum standard deviation threshold.

- min_unique:

  Integer. Minimum unique values.

## Value

Logical scalar.

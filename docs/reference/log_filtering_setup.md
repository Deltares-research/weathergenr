# Log initial setup information

Displays general information at the start of filtering.

## Usage

``` r
log_filtering_setup(n_obs, n_sim, n_realizations, sample_num, relax_priority)
```

## Arguments

- n_obs:

  Number of observations in observed series

- n_sim:

  Number of years in simulated series

- n_realizations:

  Number of realizations

- sample_num:

  Target number to sample

- relax_priority:

  Relaxation priority vector

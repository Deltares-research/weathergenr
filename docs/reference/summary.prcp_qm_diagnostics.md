# Summarize precipitation QM diagnostics

Prints a text summary for a `"prcp_qm_diagnostics"` object.

## Usage

``` r
# S3 method for class 'prcp_qm_diagnostics'
summary(object, ...)
```

## Arguments

- object:

  A `"prcp_qm_diagnostics"` object.

- ...:

  Additional arguments (unused).

## Value

Invisibly returns `object`.

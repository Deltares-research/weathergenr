# Pick a random contiguous window of years of length \`window_years\` from a year vector

Pick a random contiguous window of years of length \`window_years\` from
a year vector

## Usage

``` r
.pick_contiguous_year_window(years, window_years)
```

## Arguments

- years:

  Integer vector of available years.

- window_years:

  Integer number of contiguous years to sample.

## Value

Integer vector of selected years in chronological order.

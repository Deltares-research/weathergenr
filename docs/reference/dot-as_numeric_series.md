# Validate a numeric series input

Validate a numeric series input

## Usage

``` r
.as_numeric_series(x, name, min_n)
```

## Arguments

- x:

  Vector. Input series.

- name:

  Character scalar. Argument name (for error messages).

- min_n:

  Integer scalar. Minimum required length.

## Value

Numeric vector. Coerced to numeric.

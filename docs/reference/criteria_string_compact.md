# Compact criteria string for a filter

Creates a compact human-readable string describing current filter
criteria.

## Usage

``` r
criteria_string_compact(
  filter_name,
  bounds,
  tail_metrics,
  wavelet_active,
  wavelet_pars
)
```

## Arguments

- filter_name:

  Character filter name

- bounds:

  Bounds environment or list

- tail_metrics:

  Tail metrics list

- wavelet_active:

  Logical

- wavelet_pars:

  Wavelet parameters

## Value

Character string (compact)

# Build data for plotting

Build data for plotting

## Usage

``` r
.build_plot_data(obs_results, sim_results, variables)
```

## Arguments

- obs_results:

  Output from \`.summarize_observed_data()\`.

- sim_results:

  Output from \`.summarize_simulated_data()\`.

- variables:

  Character vector of variable names.

## Value

List of standardized data frames for diagnostic plots.
